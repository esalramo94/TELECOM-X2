# 📉 Proyecto de Predicción de Churn de Clientes

Motor analítico para **identificar clientes con alta probabilidad de cancelación** utilizando técnicas de *Machine Learning*.

## 🧠 Enfoque
- Dataset de **7,043 clientes**
- Balanceo de clases con **SMOTE**
- Comparación de modelos:
  - ✅ **Regresión Logística (modelo final)**
  - Random Forest

## 📊 Resultados Clave
- **Recall Churn:** 75%
- Identificación temprana de clientes en riesgo
- Insights accionables sobre contratos, antigüedad y precio

## 🎯 Objetivo
Pasar de una estrategia **reactiva** a una **proactiva** en retención de clientes, priorizando la detección de desertores reales.

## 🛠️ Salida del Proyecto
- Modelo predictivo listo para producción
- Recomendaciones claras de **estrategias de retención**
